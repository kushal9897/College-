### **Q2: How does your grading system ensure accuracy?**

**Answer:**
"The grading system ensures accuracy through:

1. **Weighted Formula** - Mathematically precise calculation: (Attendance × 0.10) + (Assignment × 0.20) + (Midterm × 0.30) + (Final × 0.40)
2. **Database Validation** - All scores validated before storage
3. **Audit Trail** - Every grade entry is timestamped and traceable
4. **Automated Calculation** - Eliminates human error in arithmetic
5. **Grade Boundaries** - Clear classification rules (A: 90-100, B: 80-89, etc.)
6. **PDF Generation** - Official reports with verification codes

The system has been tested with various edge cases to ensure reliability."

---

### **Q3: How secure is the password storage?**

**Answer:**
"Security is a top priority. For passwords, I've implemented:

1. **Argon2 Algorithm** - Winner of Password Hashing Competition, resistant to GPU attacks
2. **Salting** - Unique random salt for each password prevents rainbow table attacks
3. **No Plain Text** - Passwords are NEVER stored in readable format
4. **Memory-Hard Function** - Makes brute-force attacks computationally expensive
5. **One-Way Hashing** - Even database administrators cannot view actual passwords

For example, the password 'H@rdw0rk' is stored as:
'$argon2id$v=19$m=102400,t=2,p=8$...' - completely irreversible."
